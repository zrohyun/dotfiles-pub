# dotfiles-pub

Linux bootstrap for loading a minimal `~/.bashrc` block and then installing private dotfiles.

## Quick Start

```bash
curl -fsSL https://raw.githubusercontent.com/zrohyun/dotfiles-pub/main/install.sh | bash
source ~/.bashrc
dpri
```

## Scope

- Linux only
- Public repo contains only bootstrap assets (`install.sh`, `bashrc.template`)
- Private full install runs from `zrohyun/dotfiles`
